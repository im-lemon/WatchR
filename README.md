# WatchR

WatchR is a simple terminal application that can download videos from ANY * website, it works for both youtube and a provided link. It has two modes for that, Youtube and Website / Other.

It works by using yt-dlp and BeautifulSoup. It has a nice Terminal User Interface (TUI) and a command line interface (CLI).

> **Any can vary in meaning**, the website might not be compatible with our scraper.

They can be ran from the App/ directory.


# Usage

TUI (Has instructions.)

CLI:

```bash
python CLI.py -m YouTube https://www.youtube.com/link_to_video_here/
```
